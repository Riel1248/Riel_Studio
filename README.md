# Riel Studio

**The full AI pipeline inside Blender.**

Text to image, image to video, 3D generation, inpainting, upscaling — all inside Blender's node editor. Powered by ComfyUI, with every custom node supported.

No more switching between apps. Build your entire AI workflow in one place.

## Requirements

- Windows 10/11 (64-bit)
- Blender 5.0 or later
- NVIDIA GPU recommended (CPU mode available)

---

## Installation

Download **RielStudioSetup.exe** from Gumroad and run it. The installer handles everything automatically — Blender, ComfyUI, Python, PyTorch, and the Riel Studio addon.

## Roadmap

This is where Riel Studio is headed. The order may shift based on feedback.

### Now available
- Run any ComfyUI workflow inside Blender's node editor
- Every custom node supported — current and future
- Full video pipeline (Wan, LTX, AnimateDiff, etc.)
- Image generation, inpainting, upscaling, 3D mesh generation
- One-click Windows installer (Blender + ComfyUI + Python + PyTorch)
- Camera/viewport as direct ComfyUI input
- Built-in mask painting tools

### Coming next
- **Claude MCP integration** — Talk to Riel in natural language through Claude Desktop.
- **Cloud GPU** — Run workflows on cloud GPUs without owning an NVIDIA card.
- **Subscription system** — A single balance covering all costs, with full pricing transparency.
- **One-click workflow library** — Browse and install community workflows inside Blender.
